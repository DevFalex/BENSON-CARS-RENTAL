== Installation ==


This section describes how to install the plugin and get it working.


= Automatic Installation =


1. Login to Wordpress admin panel
2. Click on Plugins > Add New
3. Find the plugin by typing 'eaSYNC Booking' on the search bar
4. Click the 'Install Now' button
5. After the successful installation, click on the Activate button to start using the plugin


= Uploading the Wordpress Plugin =


1. Login to Wordpress admin panel
2. Click on Plugins > Add New
3. Click on the 'Upload Plugin' button
4. Once redirected to the plugin upload page, click the Choose File button and select the eaSYNC Booking plugin file
5. Click on the ‘Install Now’ button
6. After the successful installation, click on the Activate Plugin link to start using the plugin

== Frequently Asked Questions ==


= How can I embed booking modules on a page or post? =


To embed modules like Car Rental, go to Page or Post in the admin panel and use this shortcode: "[easync_car_code]" without double quote. 
For other modules:
[easync_hotel_code] = Hotel Booking
[easync_restau_code] = Restaurant Reservation
